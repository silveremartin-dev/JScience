Maxmind, http://www.maxmind.com/, offers a free database that identifies the country based on the IP address. http://www.maxmind.com/app/geoip_country (for the database) http://www.maxmind.com/app/java (for the API) This database is reproduced here in GeoIP.dat. iptocountry.csv is another database with the same features taken at http://ip-to-country.webhosting.info/node/view/6 Another interesting database is available at http://www.maxmind.com/download/worldcities/ although it is too big to be included here (over 33MB). It contains the following fields for approximately 2,710,000 records: City Name Accent City Name World Region Code Population (paid version) Latitude Longitude CIA world factbook from which Mondial is built on: http://www.odci.gov/cia/publications/factbook Download page of CIA world factbook: http://www.cia.gov/cia/download.html Country codes available at: http://www.davros.org/misc/iso3166.html Mondial database at: http://www.dbis.informatik.uni-goettingen.de/Mondial/
