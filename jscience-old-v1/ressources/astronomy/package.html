There are several uses for this package: You can use it to: store astronomical data compute planetary motions (or small bodies) display the solar system (using Java3D constructors) design other star systems (fake or real) A good source for astronomical data is: http://www-gsss.stsci.edu/gsc/gsc2/GSC2home.htm http://portail.imcce.fr/page.php?nav=fr/ephemerides/index.php
