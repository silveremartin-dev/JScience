Databases we support (these are meant to be the most inclusives databases): http://www.ebi.ac.uk/taxy/ http://www.cbif.gc.ca/pls/itisca/ (ITIS*ca database of taxonomic nomenclature) About the Itis database: This interface includes the ability to search the ITIS*ca database for a string match to a taxon name, which returns the ITIS Taxonomic Serial Number (TSN) for that taxon. One can then use the TSN to query ITIS*ca and find the scientific name, rank, author, synonym taxa, parent taxa, and children taxa (e.g., species within a genus), among other information. The "Itis" class is the main entry point for programmers to utilize the interface. All communication with the ITIS*ca database is accomplished using HTTP requests and XML formatted responses. See ITIS*ca for details on the ITIS database.
