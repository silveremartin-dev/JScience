Virtual Earth Data sources:

Apart from planet Earth, we could produce digital biospheres for planets whether from fiction (custom height map and texture, using for example Photoshop) or from our solar system. For such files, see:
  • http://maps.jpl.nasa.gov/
  • http://www.snyderweb.com/planets/
  • http://gw.marketingden.com/planets/planets.html
  • http://nssdc.gsfc.nasa.gov/planetary/planetfact.html

Eventually, we could model asteroids:
  • http://astronomy.swin.edu.au/~pbourke/modelling/asteroid2/
  •  http://www.space-graphics.com/

The Earth texture files are from http://earthobservatory.nasa.gov/Newsroom/BlueMarble/

Real time cloud maps avaiable from http://www.sat.dundee.ac.uk/ (and especially http://www.sat.dundee.ac.uk/pdus/XX/)

Global earth topography and bathymetry as available from:
  • http://pc12-c714.uibk.ac.at/GPLIGC/download/digital_elevation_model/etopo2/
  • http://dss.ucar.edu/datasets/ds759.3/

Files from the Global Land Cover Characteristics Data Base Version 2.0 can be found at http://edcdaac.usgs.gov/glcc/globe_int.html.
The files in use are the one from http://edcdaac.usgs.gov/glcc/tabgeo_globe.html and correspond to Geographic Projection (because the files are easier to manipulate with GTOPO30 DEMs) for the Global Ecosystems (the most precise coding, defining for example the difference between inland an ocean water). The documentation is from http://edcdaac.usgs.gov/glcc/globdoc2_0.html.

Database of geographic feature names:
  • http://earth-info.nima.mil/gns/html/index.html
  • http://geonames.usgs.gov/stategaz/index.html

Data for global population estimates and densities for 1990 and 1995 on a 2.5’ grid from http://sedac.ciesin.columbia.edu/plue/gpw/index.html?main.html&2, this data is called GPW.
The GPW grid data are available in two formats: as ArcInfo interchange files (.e00) and Binary Interleaved by Line (BIL) format files.

Hieght maps for Earth are an updated version of GTOPO30 patched with files from SRTM30. These files are rezipped after the SRTM files found at http://edcsgs9.cr.usgs.gov/pub/data/srtm/SRTM30/. We also added the original antartica files from GTOPO30 (http://edcdaac.usgs.gov/gtopo30/gtopo30.html) which could not be found in the SRTM distribution. Moreover, in order to save some disk space, we have removed the extra .dif, .jpg, .num and .std files which could not be found in the antartica files anyway. Please also read GTOPO30_Documentation, SRTM30_Documentation and SRTM30_version_history

River flows are files from the Hydro1K project at http://edcdaac.usgs.gov/gtopo30/hydro/. More files are available from there.

Geopolitical data available from http://www.dbis.informatik.uni-goettingen.de/Mondial/

Starmaps are the files from http://maps.jpl.nasa.gov/stars.html

Feature maps are Vmap0 (also known as digital chart of the world) and Vmap1 are available from http://geoengine.nima.mil/
