Terms of Use for SIL Three-letter Language Codes and their Mapping to ISO 639

In the interest of fostering the uniform identification of all the world's languages in information systems, SIL International releases for public use the system of three-letter codes used in the Ethnologue as well as their mapping to the codes of the ISO 639 standard. You are welcome to download the complete code set or the mappings as provided and incorporate the supplied tables into your own database application. The 130 codes in the range QVA to QZZ are reserved for local use, and you may assign meanings that are specific to your application to codes in this range. You are also authorized to make that application available to the public, provided that you:

  • cite SIL International and this web site (www.ethnologue.com) as the source of the information,
  • do not modify or extend the codes other than those set aside for local use, and
  • use only the data in the tables and no other data posted on this site. Rather, other information on this site should be accessed by supplying a link like the following (where XXX is an SIL code and YYY is an ISO code):
    • http://www.ethnologue.com/show_language.asp?code=XXX
    • http://www.ethnologue.com/show_iso639.asp?code=YYY

SIL International maintains the SIL language code set and the ISO 639 mappings, and intends this site to be the centralized distribution source of both in order to ensure uniformity. You are not authorized to redistribute the downloadable code or mapping tables, whether in the exact form they were obtained from this site or in a modified form you have developed, without the written consent of SIL International.
