Some of the most famous mathematicions of the 18th-century, like Abel, Gauss, Jacobi, or Weierstrass, contributed to this subject and opend a complete new field in mathematics.

We compute all elliptic functions in terms of Jacobi's theta functions, which are implemented as static functions in {@link ellipticFunctions.Elliptic}.

{@link ellipticFunctions.Jacobi} is devoted to the Jacobi's elliptic functions: sn, cn, and dn. It also includes utillity functions to compute back and forth between the different descriptions of the modulus.

An implementation for the different elliptic functions of Weierstrass, will be added soon.

All the functions have a convient and an numerical efficient front-end. Compare for example {@link #Jacobi.sn(Complex,Complex)} and {@link #Jacobi.sn(Complex,Complex,Complex[],Complex)}.
