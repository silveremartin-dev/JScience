This directory contains temporary files to merge in the code from the P. Csapo package. Don't use the files in this directory. This package should be considered as deadcode and completly removed. The aim is to have parsers VSOP87 and ELP2000 for astronomical data but not to have a 2D gui applet as part of this API. Eventually, if people want an output, they should use the 3D system from org.jscience.astronomy. Yet, this code is still useful as to debug and change the code from org.jscience.astronomy.solarsystem.ephemeris
